# `@typed-system/color`

> CSS Color related style props.

`yarn add @typed-system/color`

| Prop                      | CSS Property       | Theme Key |
| ------------------------- | ------------------ | --------- |
| `$color`                  | `color`            | `colors`  |
| `$bg`, `$backgroundColor` | `background-color` | `colors`  |
| `$opacity`                | `opacity`          | none      |
