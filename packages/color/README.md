# `@typed-system/color`

CSS Color related style props.

```shell
yarn add @typed-system/color
```

--

| Prop               | Alias | CSS Property       | Theme Key |
| ------------------ | ----- | ------------------ | --------- |
| `$color`           |       | `color`            | `colors`  |
| `$backgroundColor` | `$bg` | `background-color` | `colors`  |
| `$opacity`         |       | `opacity`          |           |
